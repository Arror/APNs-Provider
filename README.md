# APNs Provider

APNs Provider是一个Apple设备推送测试工具。方便开发者调试推送。

## 特性

a) 支持基于JWT的推送。

b) 支持基于证书的推送。

c) 支持自定义负载。

d) 支持设置推送消息优先级。

e) 支持黑暗模式。

## 使用

a) 拖拽证书文件到App。

b) 添加组织、钥匙、套装、令牌等信息。

c) 修改负载。

d) 点击底部发送按钮。

## 隐私政策

APNs Provider不会收集您的数据信息。

a) APNs Provider不会收集您的开发者、证书、负载、套装、令牌等信息，这些信息被安全的存储在macOS的Keychain中。

b) APNs Provider除了建立与苹果推送服务器的网络连接外不会建立其他的网络连接。
